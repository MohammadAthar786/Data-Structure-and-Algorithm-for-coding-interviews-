# Floyd Cycle Detection (Fast & Slow)

## When to recognize this pattern

Use this when repeated states imply a cycle and two pointers can move at different speeds to detect the loop.

Common signals:
- cycle detection
- repeated state sequence
- duplicate represented as an implicit graph
- constant extra space requirement
